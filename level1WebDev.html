Part 2 – Level 1

In this part we will learn how to add basic elements to our website, such as headings, paragraphs, and images.

To add a heading, use the h1 tag: 

<h1>Welcome to my website</h1>

You can also use h2, h3, etc. for subheadings.

To add a paragraph, use the p tag: 

<p>This is my first website</p>

To add an image, use the img tag: 

<img src="cat.jpg" alt="A cute cat">

Replace cat.jpg with your image file name and always include descriptive alt text for accessibility.

Save your file and refresh your browser to see the changes. Experiment and have fun building your website!

Now we'll learn how to use the details component!

The details element allows us to create a dropdown section without using JavaScript. Inside it, we use the summary element, which is the clickable title of the dropdown.

In our example, we created a navigation dropdown. When the user clicks "Ir a sección", a list appears with a link inside it.

That link uses an anchor with href="#hola". This means that when we click it, the browser will automatically scroll to the element that has id="hola".

At the bottom of our HTML, we created a div with id="hola". That is the section the link connects to. When clicked, the page jumps directly to that part of the website.

This is very useful for navigation menus, FAQs, hidden content, or organizing information inside a webpage. 

<body style="background-color: grey;">
    <div>
        <h1 style="color: cyan;">Welcome to my website</h1>
        <p>This is my first website</p>
        <img src="cat.jpg" alt="A cute cat">
        <details>
            <summary>Ir a sección</summary>
            <ul>
                <li><a href="#hola">Ir a Hola</a></li>
            </ul>
        </details>
    </div>

    <div id="hola">
        <p style="margin-top: 1000px;">
            HOLAAAAAAA HELLOOOOO THIS IS DIV ID hola
        </p>
       </div>
    </body>
