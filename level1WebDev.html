In this part we will learn how to add basic elements to our website, such as headings, paragraphs, and images.

To add a heading, use the h1 tag: 

<h1>Welcome to my website</h1>

You can also use h2, h3, etc. for subheadings.

To add a paragraph, use the p tag: 

<p>This is my first website</p>

To add an image, use the img tag: 

<img src="cat.jpg" alt="A cute cat">

Replace cat.jpg with your image file name and always include descriptive alt text for accessibility.

Save your file and refresh your browser to see the changes. Experiment and have fun building your website!
